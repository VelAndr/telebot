h1 **Телеграм бот для работы с системой умного дома Domoticz и не только.**
=====
h2 Ключевые особенности:
* авторизация с помощью ключевого слова, ведение базы данных пользователей
* взаимодействие с Domoticz - чтение сенсоров, изменение состояния переключателей
* по команде возможно повторное чтение блока команд конфигурации
* внутренний HTTP сервер для отсылки уведомлений всем зарегистрированным пользователям
